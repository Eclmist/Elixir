Detailed Description

A simple code timer implemented with ctime.

A code timer that can measure and print the time taken from object declaration to end of object scope

◆ EndTimer()

void Timer::EndTimer ( )

Stops the timer before the timer goes out of scope.

In the case that a process does not have its own scope or that it is beneficial to not have its own scope, this function can be used to stop the timer before the scope is closed.
